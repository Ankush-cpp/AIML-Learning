# Number Guessing Game

## Description:
A simple game where the user tries to guess a randomly generated number.

## Features:
- Random number generation
- User input handling
- Hints (Too high / Too low)
- Exit option using 'quit'

## Concepts Used:
- Loops (while)
- Conditional statements
- Exception handling
- Random module

## How it works:
- Computer selects a number between 1 and 100
- User keeps guessing until correct
- Program gives hints after each guess

## Learning Outcome:
Improved logic building and user interaction handling.
