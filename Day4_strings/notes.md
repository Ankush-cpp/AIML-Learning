# Day 4: Strings

## Topics Covered:
- String basics
- Indexing and slicing
- String methods
- String operations

## Key Learnings:
- Strings are immutable
- Indexing helps access characters
- Slicing helps extract parts of strings
- String methods like upper(), lower(), replace() are useful
- Strings are important for text processing

## Examples:

text = "Python"
print(text[0])      # P
print(text[::-1])   # nohtyP

sentence = "python programming"
print(sentence.upper())
print(sentence.replace("python", "Java"))
