# Day 8: Input Handling & Exception Handling

## Topics Covered:
- User input handling
- Infinite loop (while True)
- Conditional statements
- Exception handling using try-except

## Key Learnings:
- try-except is used to handle invalid inputs
- ValueError occurs when input cannot be converted to number
- break statement is used to exit loop

## Problem:
Take user input continuously and:
- Identify if number is positive, negative, or zero
- Exit when user types 'quit'
- Handle invalid inputs safely

## Why Important:
This concept is useful for building real-world applications where user input must be validated.
