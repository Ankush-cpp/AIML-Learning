# Chat System (Mini Project)

## Description:
A simple chat system built using Object-Oriented Programming in Python.

## Features:
- User creation
- Message sending
- Chat history display

## Concepts Used:
- Classes and Objects
- Encapsulation
- Methods
- Object interaction

## How it works:
- Users send messages
- Messages are stored in chat
- Chat history can be displayed

## Learning Outcome:
- Applied OOP concepts in a real-world scenario
- Improved understanding of class interaction
