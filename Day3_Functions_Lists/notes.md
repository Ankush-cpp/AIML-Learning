# Day 3 - Functions & Lists

## Topics Covered:
- Functions in Python
- Lists and basic operations

## Key Learnings:
- Functions help in code reusability
- return statement is used to send values back
- Lists are ordered and mutable
- append() adds element at end
- remove() deletes a specific element

## Examples:

def add(a, b):
    return a + b

nums = [1, 2, 3]
nums.append(4)
