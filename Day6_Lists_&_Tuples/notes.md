# Day X: Lists & Tuples

## Topics Covered:
- Lists in Python
- Tuple basics
- List operations
- Looping through data structures

## Key Learnings:
- Lists are ordered and mutable
- Tuples are ordered but immutable
- append() adds elements to list
- remove() deletes elements from list
- Tuples cannot be modified after creation

## Examples:

nums = [1, 2, 3]
nums.append(4)

tup = (10, 20, 30)
