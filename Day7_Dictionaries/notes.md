# Day X: Dictionaries 

This module covers the fundamentals of **dictionaries in Python**, which are used to store data in key-value pairs.

---

## Topics Covered

- Dictionary creation
- Accessing values
- Adding and updating elements
- Removing elements
- Looping through dictionary
- Built-in methods

---

## 🔍 Key Concepts

### 1. Dictionary
A collection of key-value pairs.

### 2. Keys & Values
Each key is unique and maps to a value.

### 3. Mutability
Dictionaries are mutable (can be modified).
